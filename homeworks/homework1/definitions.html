<p>
Defines a paragraph and adds a an empty space before.
<h1>
Defines the first header out of 6, and they must be used in order. Cannot put h2 before h1.
<body>
Holds all the content of your webpage, following the head tag. There is only allowed one body tag per html file.
<html>
This is the root of all the other elements and is where you declare all the other elements on the page.
<title>
Is used to describe the webpage in a few words, so that you can see it on the browser title bar.
<img>
Defines an image. Some options are to include dimensions, source link, and alternative text in this tag.
<code>
Anything in between this tag is identified as code.
<button>
Creates a clickable button that can be used to submit a form, lead to a new page, and more.
