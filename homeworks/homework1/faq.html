FAQ

What did I do to get the brackets to show up correctly?

To get brackets to show up I use &lt; and &gt; to replace the brackets in my code.

When do you use an <ol> vs a <ul>?

You use an ol for an ordered list and ul for an unordered list. Inside both ul and ol you will list the list items using the tags <li> and <li>

What does the alt attribute of an <img> do?

It provides alternate text if the user is unable to read the text, this is just a brief description of what the image describes.

How do you use an <a>?

This is the hyperlink tag to bring the user from one webpage to another. If there is no href, then it simply holds as an anchor as where a link might have or should be placed.

With correct semantics, are you allowed to have more than 1 <header> element per HTML page?

Yes, a header in the main and one in the body.

How do you indicate that a <video> can have multiple different sources?

<video controls> <source src="examplevideosource" type="video/webm"> <source src="examplevideosource" type="video/mp4"> </video>
